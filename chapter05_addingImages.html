When adding an image use the img empty tag and pass to it the attribute src with the path to the image. In my case, I saved the image in a folder within my webpage project that I called "images", so I am using a relative path to add the image in src. You can also add a title attribute with a title of the image, which is not displayed and/or the atl attribute, which can be used to describe the image. The title and atl tags seem to provide important imformation used by screen readers, which would describe a web page with visual disabilities. How cool is this?! I didn't even know this excited.
