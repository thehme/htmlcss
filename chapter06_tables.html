Markup about tables

Below are examples on how to create tables.

Simple table:

The <table> tag is used to create a table, which is then built, row by row, using the <tr> tag. In order to add data to each cell in the table, the <td> tag can be used.

For example, the following table will be a 3x3 table without any titles or labels.

Simple Table with Column Titles

Suppose that you want to add titles to the columns in your table. In order to do this, you need to use the <th> tag. You would need to add a new row with the <tr> tag and then for each column, add a <th> containing the title. If the column doesn't need a title, you can leave the <th> tag empty, but it must still be included.

Spanning Table

If you want your table to take up more than one cell in the the direction of a row or the direction of the column, then you need to use the colspan or rowspan attributes of the <td> tag. The colspan and rowspan attributes can be assigned a number string correspondig to the number of cells the data should take up, e.g. "2" for 2 cells.

Monday Tues Wed Thurs Fri
8AM Daily Meetings With Michael

It is hard to appreaciate how the rows and columns are spanning across the table, but the styling should be handled with CSS. Note: the are ways of styling tables with older code, but for newer sites, the use of CSS is prefered.

When working with a very long table, it is useful to use the <thead> and <tfoot> tags to sorround the content of <tbody>. Apparently when you do this, the header stays with the table if it overflows to beyond the sice of the screen; this way the user can scroll down and not loose the titles of the columns.
