Form with Checkbox Input

In order to create a form with check boxes, the <form> element needs to be used alon with the <input> element. However, the <input> element will require a type attribute set to "checkbox" for each option.

In addition to the type attribute, the name and value attributes are required, while the checked attribute is optional.

Like with the radion buttons, the name attribute must be the same for all options and the value attribute should be the value that will be sent back to the server when the user chooses this checkbox. If you want a checkbox to be automatically checked when the web page loads, then you can set the checked attribute to "checked"
