Form Drop Down Menu

In order to create a form that allows you to display a drop down menu, the <select> element needs to be used.

The <select> element requires a name attribute and for each option in the drop down menu, the <option> tag meeds to be used. In order words, the <option> elements need to be nested within the <select> element.

The <option> element should have a value attribute set to the value that will be sent to the server, if selected. Like with the radio buttons or check boxes, if you want any selection in the drop down menu to be auto selected when the page load, then the selected attribute can be used by setting it to "selected"
