File Input Form plus Upload Button

This functionality will require creating a <form> element that will not only have an action attribute, but also a method attribute. In all the examples, the action attribute has been omitted because the input from the user is not being sent anywhere for processing, but this should be included in a real site. As for the method attribute, it should be set to "POST", when sending information to the server for processing.

The <input> element contained in the <form> element will need to have a type attribute set to "file" in order to give the user the option to browse for an image to upload. The name attribute should also be used to name the the input of the user.

When you let users load a file, you will likely want and need users to upload their input, therefore, you will need an "upload button". To create this, you simply need to add another <input> element, with a type attribute set to "submit" and a value attribute set to something descriptive like "upload".

Submit Button

On the note of buttons, suppose that you want your users to provide their e-mail, so you can alert them about their image loading successfully. To do this, you would want a text area for the e-mail, and a submit button. As in the previous example, this functionality can be accomplished using a <form> element that will contain an <input> element. Again, the <form> element will require an action and a method attribute.

To create the text area for the user to enter their e-mail, you need to create a text input, so the <input> element will have a type attribute set to "text" and a name, set to the name you want this information to have, for example, email. The submit button will be create similar to the upload button above. You will need an <input> element with a type attribute set to "submit", a name and also value attribute. Remember that the value attribute is what the users will see in the browser, so set this to something descriptive, "Subscribe" e.g.
