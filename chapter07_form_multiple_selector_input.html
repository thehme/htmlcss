Multiple Selector Form

Suppose you want to allow the users to make multiple selections on a form. In order to do this, you can create a <form> element using the <select> element. The <select> element should be given a name via it's name attibute, which will be used to keep track of the option/s the user will select. If you want to let the user see more than one option from the list of options, the size attribute can be adjusted. If you want the user to be able to select more than one option, you can do this by setting the <select>'s multiple attribute to "multiple".

For each option the user has available, you should create a new <option> element and assign a value to its value attribute. If you want any of these options to be pre-selected when the user loads the page, set its selected attribute to "selected". The option the user will see in the browser corresponds to the text you place between each option's opening and closing tags.
