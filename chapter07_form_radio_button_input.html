Form with Radio Buttons

Again, in order to create a form, the <form> element needs to used in conjunction with the <input> element

For each radio button, there should be an <input> element, with the following attributes:

  • type set to "radio"
  • name set to the same value, e.g. "option", such that once the user makes a choice, the radio button value will be assigned to this name attribute.
  • value should be set to the value that will be sent to the server when the particular radio button is selected.
  • required can be used as well, if you want to have a radio button checked by default. This attribute should be assigned the value "checked" in being used.
