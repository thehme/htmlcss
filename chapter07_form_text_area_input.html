In conjunctions with the <form> tag, the <texarea> can be used to create a text area for users to enter text. There are the cols and rows attributes that can be used to re-size the text area, but CSS is prefered for controlling the hight and width of the text area box.

An interesting aspect of the <texarea> element is that unlike the <input> element, which is an empty element and doesn't need the closing tag, the <texarea> element does need a closing tag.

Furthermore, the <texarea> element takes in the name, cols, and rows attributes, but like with the <input> element, CSS is prefered for styling.
