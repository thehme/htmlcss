Form using text input

A form allows a web page to allow input from the user in different formats of the <input> tag. The <form> tag is used to create this element and accepts the action, method, and the id attributes. The id attribute is necessary for styling and backend programming related tasks.

With regards to the method attribute, this refers to how the form will be sent, either using the POST or GET method. In general terms, the POST method is used to add, delete, or edit information, which the GET method is used to retrieve information.

The <input> accepts a type, name, and maxlength attributes, as well as a size attribute, which should not be used since CSS is prefered for this.
